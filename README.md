# Учебный проект: портфолио на Vite + Bulma + Sass

## Сбалансированный стек (по слоям)
1. Runtime: `node` — среда выполнения для инструментов сборки.
2. Package manager: `npm` или `yarn` — установка зависимостей.
3. Bundler / dev-server: `vite` — быстрый запуск и production-сборка.
4. Препроцессор: `sass` — кастомизация и удобные переменные.
5. UI-фреймворк: `bulma` — готовая сетка и компоненты.
6. Альтернатива: `webpack` — классический сборщик (в лекции сравниваем подходы).

Порядок этого перечисления помогает видеть, на каком уровне инструментальном и технологическом решается каждая задача.

## Быстрый старт (выберите один менеджер)

### npm
```bash
npm install
npm run dev
```

### yarn
```bash
yarn

yarn dev
```

Production-сборка и предпросмотр:
```bash
npm run build
npm run preview
```

## Структура
- `index.html` — каркас страницы и секции портфолио.
- `src/main.js` — логика бургер-меню, reveal-анимаций и год в футере.
- `src/styles/` — Sass-переменные, базовые стили, компоненты, утилиты.
- `practice/` — материалы для студентов и гайды по технологиям.
- `docs/` — архитектура, дизайн и developer-документация.

## Документация
