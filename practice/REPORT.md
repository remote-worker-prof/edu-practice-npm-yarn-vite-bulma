## 4. Ход работы (по шагам)

### 4.1 Установка зависимостей
Команды (использовать **один** менеджер):

npm:
```bash
npm install
```

yarn:
```bash
yarn
```

**Результат:** создан файл lock (`package-lock.json` или `yarn.lock`).

### 4.2 Запуск dev‑сервера
```bash
npm run dev
```
или
```bash
yarn dev
```

**Результат:** сайт доступен в браузере, изменения применяются без перезапуска.

### 4.3 Кастомизация Bulma через Sass
В файле `src/styles/_variables.scss` изменены минимум 3 переменные.

Пример:
```scss
$primary: #c65a3a;
$body-background-color: #f3efe7;
$family-primary: "Space Grotesk", sans-serif;
```

**Результат:** цвета и типографика изменились по всему сайту.

### 4.4 Создание кастомных блоков
Добавлены минимум 2 собственных блока (например, карточки проектов, секция «процесс», контактный блок).

**Результат:** блоки визуально отличаются от стандартных компонентов Bulma.

### 4.5 Проверка адаптивности
Проверены мобильные размеры в DevTools.

**Результат:** нет горизонтальной прокрутки, меню работает, текст читаем.

### 4.6 Production‑сборка
```bash
npm run build
npm run preview
```

**Результат:** сборка без ошибок, preview показывает итоговый сайт.
