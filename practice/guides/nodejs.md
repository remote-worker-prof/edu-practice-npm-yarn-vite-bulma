# Node.js — гайд для студентов

## Что это
Node.js — среда выполнения JavaScript вне браузера. В учебных проектах она нужна для запуска инструментов сборки и dev‑сервера, а не для работы сайта на сервере.

## Роль в этом проекте
- Запуск Vite dev‑сервера.
- Сборка production‑версии.
- Установка зависимостей через npm или yarn.

## Когда применять
- Любой фронтенд‑проект со сборкой (Vite/Webpack/Parcel).
- Локальная разработка, где нужен CLI‑инструмент.

## Как проверить установку
```bash
node -v
npm -v
```
Ожидается версия Node.js LTS (например, `v20.x` или новее).

## Типовые ошибки
- Старая версия Node.js → сборка не запускается или падает.
- Разные версии Node.js у команды → разное поведение.
- Node.js установлен, но не в `PATH` → команды не найдены.

## Мини‑задание
1. Проверьте версии `node` и `npm`.
2. Зафиксируйте версии в отчете.

## Мини‑чеклист
- Установлена актуальная LTS‑версия Node.js.
- Команды `node -v` и `npm -v` работают из терминала.
- Версии записаны в отчет.
