# Vite — гайд для студентов

## Что это
Vite — современный сборщик и dev‑сервер. Основная идея: быстрый запуск разработки и быстрая сборка.

## Зачем в этом проекте
- Мгновенный dev‑сервер.
- Production‑сборка через `vite build`.

## Основные режимы
- `dev` — разработка с hot‑reload.
- `build` — сборка оптимизированных файлов.
- `preview` — локальный просмотр собранной версии.

## Минимальные команды
```bash
npm run dev
npm run build
npm run preview
```

## Проверка результата
- Dev‑сервер открывается в браузере.
- Изменения в коде применяются без перезапуска.
- `npm run build` создаёт папку `dist/`.

## Типовые ошибки
- Запуск `vite build` без установки зависимостей.
- Открытие `index.html` напрямую вместо dev‑сервера.
- Неверный порт или занятый порт.

## Мини‑задание
1. Запустите `npm run dev`.
2. Измените текст в `index.html` и убедитесь, что обновление видно в браузере.

## Мини‑чеклист
- Dev‑сервер работает.
- Сборка проходит без ошибок.
- Preview показывает итоговый сайт.
