# Yarn — гайд для студентов

## Что это
Yarn — альтернативный менеджер пакетов для Node.js. Работает похоже на npm, но имеет свой lock‑файл.

## Как установить (если yarn нет)
Рекомендуемый способ — через Corepack (идет вместе с Node.js):
```bash
corepack enable
corepack prepare yarn@stable --activate
```

Проверка установки:
```bash
yarn -v
```

Если Corepack недоступен, используйте npm:
```bash
npm i -g yarn
```

## Зачем в этом проекте
- Установка зависимостей.
- Запуск скриптов Vite.

## Когда выбирать yarn
- Если yarn уже используется в группе/команде.
- Если преподаватель выбрал yarn как стандарт.

## Минимальные команды
```bash
yarn
yarn dev
yarn build
yarn preview
```

## Где фиксируются версии
- `yarn.lock` — точные версии зависимостей.

## Проверка результата
- Появилась папка `node_modules/`.
- Создан `yarn.lock`.
- `yarn dev` запускается без ошибок.

## Типовые ошибки
- Одновременное использование npm и yarn.
- Отсутствие `yarn.lock` после установки.
- Запуск `yarn build` без `yarn`.

## Мини‑задание
1. Установите зависимости через yarn.
2. Запустите dev‑сервер.
3. Зафиксируйте результат в отчете.

## Мини‑чеклист
- Используется только yarn.
- Есть `yarn.lock`.
- Скрипты выполняются без ошибок.
