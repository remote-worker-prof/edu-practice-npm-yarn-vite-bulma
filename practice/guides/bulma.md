# Bulma — гайд

## Что это
Bulma — CSS‑фреймворк без JavaScript. Дает сетку, готовые компоненты и набор классов.

## Роль в этом проекте
- Быстрое построение структуры страницы.
- Стандартизированные компоненты (кнопки, секции, карточки).
- Кастомизация через Sass‑переменные.

## Мини‑теория
Bulma — это набор CSS‑классов. Вы не пишете JS для компонентов, а применяете классы к разметке. Поведение интерфейса добавляется отдельно (в нашем проекте — в `src/main.js`).

## Пример
```html
<div class="columns">
  <div class="column">Колонка 1</div>
  <div class="column">Колонка 2</div>
</div>
```

## Типовые ошибки
- Ожидание встроенного JS.
- Неправильный порядок Sass‑импортов.
- Использование классов, которых нет в Bulma.

## Как понять, что всё ок
- Сетка работает (колонки выравниваются).
- Кнопки и карточки выглядят по‑булмовски.
- Изменение переменных Sass влияет на все компоненты.

## Короткий вывод
Bulma ускоряет верстку и дает стабильную сетку, а Sass позволяет сделать стиль уникальным.
