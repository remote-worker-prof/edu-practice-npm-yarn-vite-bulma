# Bulma — гайд для студентов

## Что это
Bulma — CSS‑фреймворк без JavaScript. Дает сетку, компоненты и модификаторы классов.

## Зачем в этом проекте
- Быстрое построение сетки и карточек.
- Базовые компоненты: кнопки, навигация, секции.
- Лёгкая кастомизация через Sass.

## Как использовать
- Подключать классы Bulma в `index.html`.
- Настраивать тему через `_variables.scss`.

## Типовые ошибки
- Ожидание встроенного JS (у Bulma его нет).
- Неверный порядок импортов Sass.
- Использование классов, которых нет в Bulma.

## Проверка результата
- Сетка Bulma работает (колонки выравниваются).
- Кнопки и теги имеют правильный стиль.
- Переменные темы влияют на весь интерфейс.

## Мини‑задание
1. Сделайте блок с двумя колонками.
2. Добавьте кнопку с модификатором `is-primary`.

## Мини‑чеклист
- Используются классы Bulma.
- Тема меняется через Sass‑переменные.
- Каркас страницы остаётся адаптивным.
